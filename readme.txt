Fetches and displays weather data from a local Raspberry Pi station.

== Description ==
Integrates with elewin/pi-weather-station repo to pull data via API and show in WordPress.

== Installation ==
1. Upload to /wp-content/plugins/.
2. Activate in Plugins menu.
3. Add API key in Settings > Weather API.
4. Use shortcode [weather_display] or widget.
